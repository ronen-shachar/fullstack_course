Sample Manage Students Application

## Create venv
```
python -m venv venv
```

## Create venv
On Windows
```
.\venv\Scipts\Activate
```

## Activate the venv
On Linux/MacOS
```
./venv/bin/activate
```

## Install required packages
```
pip install flask flask_cors
```

## Run the application
```
python server.py
```
